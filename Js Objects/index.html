JavaScript Objects (data type)

a JavaScript object is a collection of named values.

object values are written as name : value pairs (name and value seprated by a colon).

the naemd value, in JavaScript objects, are called properties,

it is a common practice to declare object with the const keyword.

in real life, a car is an object, has properties like weight and color, and mathods like start and stop:

all cars have the same properties, but the property values differ from car to car.

all cars have the same methods, but the methods are preformed at different times.

the simplest way to create an object is by using and object literal.

spaces and line breaks are not importent, an object definition can span multiple lines.
