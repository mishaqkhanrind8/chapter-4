JavaScript Alert, Confirm, and Prompt

alert box

An alert is a box that pops up to give the user a message.

alert("thanks for your input");

This page says

Thanks for your input

prompt box

JavaScript Prompt Box can be used when we want to get some user input.

Following is the syntax for the JavaScript Prompt box:

var userName = Prompt("Please enter your name:");

You can provide a default value for the prompt in case the user cancels or leaves it blank.

var userName = prompt("Please enter your name:", "Guest");

This page says

Please enter your Name!

confirm box

The confirm() box displays a dialog box with a message to the user and two buttons: "OK" and "Cancel".

It's typically used when you need to ask the user to confirm or cancel an action, such as deleting something or proceeding with a critical operation.

The user can click "OK" to confirm the action or "Cancel" to cancel it.

var confirmation = confirm("are you sure you want to delete this item?");

Are you sure you want to delete this item?
