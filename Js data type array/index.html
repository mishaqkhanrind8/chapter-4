js data type array

an array is a special variable, which can hold more than one value.

arrays are a special type of objects. the typeof operator in JavaScript returns "object" for arrays.

the following code declares (creates) an array called cars, containing three items (car names)

const cars = {"saab" , "volvo", "BMW"};

if you have a list of items (a list of car names, for example), storing the cars in single variables could look like this:

  • let car-1 = "saab";
  • let car-2 = "volvo";
  • let car-3 = "BMW";

however, what if you want to loop through the cars and find a specific one? and what if you had not 3 cars, but 300?

the solution is an array

an array can hold many values under a single name, and you can access the values by referring to an index number.

converting on array to a string

the JavaScript method toString() converts an array to string of (comma seprated) array values.

const students = ["asad", "sajid", "babar"];
let x = students.toString();
//result : x = "asad, sajid, babar" (string)

the join() method also joins all array elements into a string.

it behaves just like toString() but in addition you can specify the seprator:

const fruits = ["banana", "orange", "apple", "mango"];
console.log= fruits.join(" * ");

the pop() method

the pop() method removes the last element from an array

const fruits = ["banana", "orange", "apple", "mango"];
fruits.pop();

the pop() method returns the value that was "popped out".

fruit.pop(); //returns Mango

the push() method

the push() method adds a new element to an array (at the end).

const fruits = ["banana", "orange", "apple", "mango"];
fruits.push("peach");

console.log = fruits;
result : Banana, Orange, Apple, Mango, peach

the shift() method

the shift() method removes the first array element and "shifts" all other elements to a lower index

shift() is similiar to pop(). but works on the first element instead of the last.

const fruits = ["banana", "orange", "apple", "mango"];
fruits.shift();

the shift() method returns the value that was "shifted out",

fruits.shift(); //returns Banana

the unshift() method

the unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements;

const fruits = ["banana", "orange", "apple", "mango"];
fruits.unshift("peach");

console.log= fruits;
result: peach, banana, orange, apple, mango.

the delete() method

the delet() method is used to remove an element from an array.

note: using delete() leavs undefined holes in the array. use pop() or shift() instead.

const fruits = ["banana", "orange", "apple", "mango"];
delete.fruits[2];

console.log= fruits;
result: peach, banana, empty, apple, mango.

concat () Method

the concat() method creates a new array by merging (concatenating) existing arrays.

the concat() method does not change the existing arrays. it always returns a new array.

const arr1 = {11, 12 ,13};

const arr2 = {14, 15 ,16};

const newarr = arr1.concat(arr2);

result: 11, 12, 13, 14, 15, 16

flattening an array

flattening an array is the process of reducing the dimensionality of an array.

flattening is useful when you want to convert a multi-dimensional array into a one-dimentional array.

the flat() method creates a new array with sub-array elements conceatenated to a specified depth.

const myarr = [[1,2],[3,4],[5,6]];
const newarr = myarr.flat();
result: 1, 2, 3, 4, 5, 6

the splice() method is used to insert and remove elements at the same time in between the array.

the slice() method

the slice() method returns a new array containing a portion of the original array. based on the start and end index provided as arguments.

syntax

array.slice (startindex, endindex);

when the slice() method is given two arguments. it selects array elements from the start argument, and up to (but not include) the end argument.

const arr = [1, 2, 3, 4, 5, 6, 7, 8];
arr.slice(4, 7);
// retuns 5, 6, 7

if the end argument is omitted, the slice() method slices out the rest of the array.

the slice() method creates a new array.

the slice() method does not remove any elements from the source array.

the indexOf() method

the indexOf() method searches an array for an element value and returns its position.

array.indexOf(item,start)

item (required); the item to search or,

start (Optional); where to start the search negative values will start at the given position counting from the end, and search to the end.

const fruits = ["banana", "orange", "apple", "mango"];
fruits.indexOf("mango") // returns 3

array.indexOf() returns -1 if the item is not found.

if the item is present more than once. it returns the position of the first occurrence.

const fruits = ["banana", "orange", "apple", "mango", "banana"];
fruits.indexOf("peach") // returns -1
fruits.indexOf("banana") // returns 0
fruits.indexOf("banana", 2) // returns 4

the lastIndexOf() method

array.lastIndexOf() is the same as array.indexOf() but returns the position of the last occurrence of the spacified element.

sorting and reversing

the sort() method sorts an array alphabetically:

const fruits = ["banana", "orange", "apple", "mango", "banana"];
fruits.sort()
// result : 'apple' , 'banana' , 'banana', 'mango', 'orange'

the reverse() method reverses the elements in an array;

const arr = [1,2,3,4,5,6,7,8];
arr.reverse();
// returns: 8,7,6,5,4,3,2,1

numeric sort

by default, the sort() function sorts values as trings.

this works well for strings ("apple" comes before "banana").

if numbers are sorted as strings, "25" is bigger than "100". because "2" is bigger than"1".

you can the numeric sorting problum by providing a compare function

const points = [40, 100, 1, 2,25, 10];
points.sort(function(a, b){return a - b});

use the same trick to sort an array descending:

points.sort(function(a, b){return b - a});

compare function

the compare function compares all the values in the array, two values at a time (a, b).

find min or max with sort()

after you have sorted an array, you can use the index to obtain the hightest and lowest values.

const points = [40, 100, 1, 2,25, 10];
points.sort(function(a, b){return a - b});

now points[0]contains the lowest value and points [points.length-1] contains the highest value.
