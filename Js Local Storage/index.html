JavaScript Local Storage

local storage

local storage is a way to store data on the user's browser.

it's like a small database that lives inside your web browser. This storage persists even after you close the browser.

local storage is a window object property, which makes it a global object that can interact with and manipulate the browser window.

window.localStorage

The localStorage mechanism is available via the window.localStorage property.

Window.localStorage is part of the window interface in JavaScript, which represents a window containing a DDM document.

THe window interface features a wide rang of functions, constructors, objects, and namespaces.

WIndow.localStorage is a read-only property that returns a refernce to the local storange abject used to store data that is only accessible to the origin that created it.

when to use localStorage?

Put simply, localStorage is used for storing and retrieving data, While you can store small amounts of data with localStorage, it's not suitable for large datasets.

localStorage is accessible to anyone who uses the device, so you shouldn't use it to store sensitive information.

You can use it to store user preferences, like language or theme.

You can also use it to cache data if you use it frequently.

localStorage can store form data that won't be lost if the user closes the browser.

if you have an application that requires you to log in, localStorage can be used to keep your session data.

You can remain logged in even after closing and reopening the browser.

where is localStorage stored?

LocalStorage is stored within the browser itself.

Each browser (e.g., Chrome, firefox, Safari) has its own implementation and storage location for this data.

On a lower level, localStorage is seved to the file system of your device.

For instance, in Chrome, it's stored in the user's profile directory under a specific file path, which varies depending on the operating system.

WIndows: C:\User\<YourUsername>\AppData\Local\Google\Chrome\User Data\Default\Local Storage

macOS: -/Library/application Support/Google/Chrome/Default/Local Storage

Linux: -/config/google-chrome/Default/Local Storage

key features

Data stored in local storage presists even after the broser is closed or the computer is restored.

Each webiste can store up to 5MB of data in local storage. which is sually enough for stroing user preferences. settings, or small omounts of data.

Local storage can only store strings.

This means if you need to store values like objects or arrays, you first need to get a string representation of the value. You do this using the JSON.stringify() method.
