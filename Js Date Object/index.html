JavaScript is often used in web development for tasks involving dates and times.

this includes creating features like calenders,train schedules, appointment setters, and interfaces that display relevant times based on the user's timezone.

to achieve these objectives and more, JavaScript comes with the built in date object and related methods

creating date objects

Date object are created with the new date() constructor:

there are 9 ways to create a new date object:

new Date()
new Date(date string)
new Date(year, month)
new Date(year,month,day)
new Date(year,month,day,hours)
new Date(year,month,day,hours,minutes)
new Date(year,month,day,hours,minutes,seconds)
new Date(year,month,day,hours,minutes,seconds,ms)
new Date(milliseconds)

date objects

new Date() creates a date object with the current date and time:

const d = new Date();
console.log(d); // thu mar 07 2024 10:50:45 GMT+0500 (Pakistan Standard Time)

new Date(date string) creates a date object from a date string:

const d = new Date("October 13, 2014 11:13:00");
console.log(d); // mon oct 13 2014 11:13:00 GMT+0500 (Pakistan Standard Time)

new Date(year, month, ...) creates a date object with a specified date and time.

7 numbers specify year, month, day, hour, minute, second, and millisecond (in that order);

const d = new Date(2018, 11, 24, 10, 20, 30, 0);
console.log(d); // mon dec 24 2018 10:20:30 GMT+0500 (Pakistan Standard Time)

JavaScript counts months from 0 to 11. (January = 0, December = 11)

Specifing a month heigher than 11, will not result in an error but add the overflow to the next year,

JavaScript stores Dates as milliseconds.

JavaScript stores dates as number of milliseconds since January 01, 1970.

zero time is January 01, 1970 00:00:00 UTC.

One day (24 hours) is 86 400 000 milliseconds.

Now the time is: 17097973930345 milliseconds past January 01, 1970.

new Date(milliseconds) creates a new date object as milliseconds plus zero time;

01 January 1970 plus 100 000 000 000 milliseconds is :

const d = new Date (100000000000);
console.log(d); // Sat Mar 03 1973 14:46:40 GMT+0500 (Pakistan Standard Time)

January 10 1970 minus 100 000 000 000 milliseconds is :

const d = new Date(-100000000000);
console.log(d); // Mon Oct 31 1966 19:13:20 GMT+0500 (Pakistan Standard Time)

date methods

when date object is created, a number of methods allow you to operate an it.

date methods allow you to getand set the year, month, day, hour, minute, second,and millisecond of date object using either local time or UTC (universal, or GMT) time

date get methods

the getFullYear() method returns the year of a date as a four digit number:

the getMonth() method returns the month of a date as a number (0-11):

the getDate() method returns the day of a date as a number (1-31)

const d = new Date();
d.getFullYear(); // 2024
d.getMonth(); // 10
// to get the correct month number , you must add 1

the getDay() method returns the weekday of a date as a number (0-6).

in JavaScript, the first day of the week (day 0) is sunday.

the getTime() method returns the the number of milliseconds since January 1. 1970.

const d = new Date();
d.getDay(); // 5
d.getTime(); // 1709l894245145

the getHours() method returns the hours of a date as a number (0-23).

the getMinutes() method returns the minutes of a date as a number (0-59).

the getSeconds() method returns the seconds of a date as a number (0-59)

the getMilliseconds() method returns the milliseconds of a date as a number(0-999).

const d = new Date();
d.getHour();
d.getMinutes();
d.getSeconds();
d.getMilliseconds();

UTC date get methods

UTC methods use UTC time (Coordinated Universal Time)

UTC time is the same as GMT (Greenwich Mean Time)

methoddescription
getUTCDate()Returns the UTC date
getUTCHours()Returns the UTC hour
getMinutes()Returns the UTC minutes

date set methods

set date methods let you set date values (years, months, days, hours, minutes, seconds, milliseconds) for a date object

the setFullYear() method sets the year of a date object, in this example to 2020.

const d = new Date();
d.setFullYear(2020);

the setFullYear() method can optionally set month and day:

d.setFullYear(2020, 11, 3);
console.log(d) // Thu Dec 03 2020 15:46:11 GMT+0500 (Pakistan Standard Time)
