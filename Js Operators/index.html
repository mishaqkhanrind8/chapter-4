arithmetic operators

arithmetic operators perform arithmetic an numbers (literals or variables).

let x = 16 + 4 // literals
let x = a + b ; // veriables

the numbers (in an arithmetic operators) are called operands.

the addition operator (+) adds numbers.

the subtraction operator (-) subtracts numbers.

the multiplication operator (*) multiplies numbers.

the division operator (/) divides numbers.

the modulus operator (%) returns the division remainder.

let x = 10 % 3;
// = 1

the increment operator (++) increments numbers

let x = 5;
x++; // now x value is 6

the decrement operator (--) decrements numbers.

let x = 5;
x--; // now x value is 4

the exponentiation operator (**) raises the first aperand to the power of the second operand.

let x = 5 ** 2;
// 52

++ increment , --decrement operator can be used as prefix or postfix.

prefix operator first increase/decrease the value in variable and then assign result to other variable.

postfix operator first assign the value in other value then increase/decrease value in actual variable.

operators precedence

operator precendence in JavaScript determines the priority of operators in an operation.

it determines which operators have higher precedence than others, and thus tells us the order to perform a given mathematical expression. consider following expression.

let x = 5 + 2 * 3 - 2 / 2 ;

the evaluation of above expression depends on BODMAS rule.

BODMAS rule

B- bracket first

O- order i.e powers and square roots etc

DM- division and multiplication (left to right)

AS- addition and subtraction (left to right)

let x = 5 + 2 * 3 - 2 / 2;
// result wil be 10

var a = 5 + 2* (3 - 2) / 2; // result 6

5-2 with brackets will be evaluated first, result 1.

2* result of (3 -2) so 2 * 1. result 2.

result of 2*(3-2) divide by 2 so 2 / 2. result 1.

5 + result of 2* (3-2)/ 2. so 5 + 1, result 6
